Présentation du projet

Constat sur le développeur

  • Fainéant
  • Grand consommateur de café
  • Sort peu de sa zone de confort

  • Réalisation d'un système de gestion de cafetière

  • À distance

  • Cafetière physique (parce que le café virtuel ...)

Explication des différentes classes

Client

  • Composé d'une View et d'un ClientNetController

  • Execute une commande défini

  • bin/cpc client webcoffee 80 start

Server

  • Composé d'un Driver et d'un ServerNetController

  • Ecoute sur le port passé en paramètre

  • bin/cpc server webcoffee 80

NetController

  • Gère les interactions réseaux (lib boost)

  • Réagi aux Requete/Réponse

  • Génère du HTTP (plus ou moins...)

PotState / Driver

  • Gère l'état du système

  • PotState - virtuel / Driver - physique

  • Utilisation de scripts python
